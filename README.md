# book api

**use GOODREADS API to get details of specific books**

* Inside:
  * API
  * requests
  * testing
  * xml response

## How to use:

**Requirements**
```
pip install requests
```

**To run the main file:**
```
python main.py
```
Enter your url when it asks for input

**To run the test file:**
```
python test_p.py
python test_n.py
```
